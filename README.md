## 퍼뜩

퍼뜩은 내 목소리로 아침에 깨워주는 알람 앱이에요!

기획부터 배포까지 4일 동안 바이브코딩으로 만든 앱이고, 현재 일주일만에 84명의 다운로드 수를 기록했어요!<br/>
iOS 26을 타겟으로 AlarmKit을 적용해 더 큰 소리로 목소리 알람을 울리기 위해 업데이트하고 있어요!
