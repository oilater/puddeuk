## 퍼뜩

퍼뜩은 내 목소리로 아침에 깨워주는 알람 앱이에요!


기획부터 배포까지 4일 동안 바이브코딩으로 만든 앱이며, 현재 운영하고 있어요.

퍼뜩은 iOS 26부터 이용할 수 있어요. 원래 모든 버전에서 이용 가능했지만, 더 확실하고 안정적인 알람 기능을 제공하기 위해 지원 대상을 좁혔어요.


iOS 정책에 따라 계속해서 업데이트 할 예정이며, iOS 26 이하 유저분들도 이용할 수 있도록 확대할 계획입니다.

## Update Log

### v 1.1.0

- iOS 26 AlarmKit 기반으로 더 정확하고 큰 알람을 들을 수 있어요
- 무음, 방해금지 모드에서도 알람이 울려요
- 반복 날짜를 더 편하게 선택할 수 있어요
- 스누즈 기능을 추가했어요
- Live Activity, Dynamic Island를 적용했어요
- 따로 녹음을 하지 않으면 기본 시스템 알림이 울려요

### v 1.0.0

- 내 목소리를 알람음으로 설정해요!
- 다시 알림, 활성화/비활성화 등 기본적인 알람 앱의 기능을 모두 제공해요.
- 백그라운드에서도, 앱이 꺼져 있어도 무조건 목소리 알람이 울려요
- 방해금지 > 수면 모드에서도 알람이 울려요(퍼뜩 알림 허용)
- 무음모드가 켜져 있을 때는 알람이 안 들려요
